72 // Expands 'buf' underlying buffer to the given new size 'newSize'.
73 //
74 // If 'buf' did not own the underlying buffer, a new buffer is
75 // created and data copied over. The original data is now detached.
76 // The underlying buffer is now owned by 'buf' and will be freed when
77 // the natsBuffer is destroyed.
